# form

Composable form layout — rows, columns, labels, help text, error text.

## Usage

```html
<form class="domi-form">
  <div class="domi-form__row">
    <div class="domi-form__col">
      <label class="domi-form__label">Name</label>
      <input class="domi-input">
    </div>
  </div>
</form>
```
