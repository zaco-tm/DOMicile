# card

Glass surface for grouped content.

## Usage

```html
<div class="domi-card">
  <div class="domi-card__header">Title</div>
  Body content
</div>
```

## Variants

- `domi-card` — default (16px padding, 16px radius)
- `domi-card--sm` — compact (8px padding, 8px radius)
- `domi-card--lg` — roomy (24px padding)
